# Open Source

FormCN is proudly open source and community-driven. We believe in building tools that empower developers to create better software, faster.

## Our Philosophy

*   **Transparency**: All our code is public and available for inspection.
*   **Community First**: We value contributions, feedback, and ideas from the community.
*   **Modern Stack**: Built with the best modern web technologies (Next.js, Shadcn UI, Better Auth).

## Contributing

We welcome contributions of all kinds! Whether it's fixing bugs, adding new features, improving documentation, or creating new templates.

### Getting Started

1.  **Fork the repository** on GitHub.
2.  **Clone your fork**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/formcn.git
    cd formcn
    ```
3.  **Install dependencies**:
    ```bash
    pnpm install
    ```
4.  **Start the development server**:
    ```bash
    pnpm dev
    ```

### How You Can Help

*   **Report Bugs**: Found an issue? Open a ticket on our [GitHub Issues](https://github.com/abdullah-dev/formcn/issues).
*   **Suggest Features**: Have an idea? Start a discussion or submit a feature request.
*   **Submit Pull Requests**: Implement a fix or a new feature and send us a PR.
